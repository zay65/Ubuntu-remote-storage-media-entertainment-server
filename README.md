# Ubuntu-remote-storage-media-entertainment-server
<h3>Repo of my Ubuntu (24.04.4 LTS) home server where I can store and share files/pictures with Samba and watch movies/shows with Jellyfin from anywhere in the world using SSH and Tailscale.</h3>

<div align="center"> After headless Ubuntu install, I SSH into my server from my windows laptop.</div>
